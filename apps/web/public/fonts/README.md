# Fonts

Place the **Neue Machina** web fonts here for the `font-machina` Tailwind class
(used by the landing hero heading):

- NeueMachina-Regular.woff2
- NeueMachina-Medium.woff2
- NeueMachina-Bold.woff2

Neue Machina is a paid Pangram Pangram font — buy/download it from
https://pangrampangram.com/products/neue-machina and export .woff2 weights.

Until the files are present, `font-machina` falls back to Sora automatically.
